# Roadmap

## Phase 0 - Product boundary

- isolate local runtime from product repo
- define allowlist export rules
- define sanitization and secret blocking
- standardize docs, changelog, roadmap, templates

## Phase 1 - Product core

- control plane API
- automation catalog and execution model
- tenant bridge bootstrap
- product contracts and SDK

## Phase 2 - Durable orchestration

- Temporal workflows for release flows and long-running jobs
- execution checkpoints
- retries, compensations, and approvals
- audit-ready state transitions

## Phase 3 - Multi-tenant platform

- tenant-aware RBAC
- per-tenant configuration and secrets
- tenant usage isolation
- product-grade onboarding

## Phase 4 - Cloud-native delivery

- Helm chart
- Argo CD GitOps
- autoscaling, probes, and network policies
- observability stack

## Phase 5 - Commercial readiness

- packaging tiers
- licensing model
- demo environment
- support runbooks
- upgrade and migration policy
